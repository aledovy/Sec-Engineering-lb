# Sec-Engineering-lb
This repository consists of file, scripts and explanation needing to set up and maintain Wazuh lab.

I have included steps for VM setup and also for Docker setup one mimicing more production enviroment and other based on devops.

## Requirements

- **Hardware**:
  - Computer with at least 16GB RAM and dual-core processor
- **VM/ISO Image**:
  - Wazuh OVA Image
  - Windows 10 Machine OVA image
  - Ubuntu Server 22.04 Image
  - Kali Linux(Attacker Machine) OVA image

- **Tools Used**:
  - Wazuh
  - VMware
  - Linux Ubuntu VM
  - Docker
